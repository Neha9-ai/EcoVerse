# EcoVerse AI

EcoVerse AI is a polished landing page concept for an intelligent climate action platform. It combines:

- an AI-style assistant experience
- a carbon footprint calculator
- feature highlights for satellite monitoring, tree planting guidance, and waste recognition
- a modern product-style presentation for hackathons or portfolio demos

## Run locally

Open [index.html](index.html) directly in a browser, or serve the folder with a simple static server:

```bash
cd d:/EcoVerse-AI
python -m http.server 8000
```

Then visit http://localhost:8000.
